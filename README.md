# HelpDesk System
System for handling customer tickets created using PHP, with Laravel &amp; Filament, and MySQL for the database.

### Features
Allows you to create, update, delete, and track customer support tickets. 
Each ticket has a unique ticket number, creator, subject, status, priority, and created time.
As an admin, you can browse through tickets, edit, or delete any of them, as well as leave comments and view their edit history.
